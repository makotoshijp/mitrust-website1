# MIトラスト合同会社 コーポレートサイト

経営課題に当事者として参画し、売上・利益拡大を実現する戦略パートナー「MIトラスト合同会社」の公式ウェブサイトです。

## 🌟 プロジェクト概要

MIトラスト合同会社は、企業の変革支援（売り物・売り方・売る人）を専門とするコンサルティング会社です。このウェブサイトは、同社のサービス、実績、企業理念を経営層に向けて効果的に伝えるために設計されたプロフェッショナルなコーポレートサイトです。

### 主な特徴

- **信頼感のあるデザイン**: 経営層をターゲットにした洗練されたプロフェッショナルなビジュアル
- **データドリブンな訴求**: 具体的な数値実績（+10～15%売上成長、+20%CxO面談増加）を強調
- **明確な価値提案**: 「提言で終わらない」「設計→実装→定着」の差別化ポイントを明示
- **完全レスポンシブ対応**: デスクトップ、タブレット、スマートフォンで最適な表示
- **スムーズなUX**: ナビゲーション、スクロールアニメーション、フォームバリデーション

### コードのカスタマイズ

#### 色の変更
`css/style.css`の`:root`セクションでCSS変数を変更：
```css
:root {
    --primary-color: #1a56db;  /* メインカラー */
    --secondary-color: #0e7490; /* セカンダリカラー */
    --accent-color: #f59e0b;    /* アクセントカラー */
}
```

#### コンテンツの更新
`index.html`で各セクションのテキストを直接編集可能

#### フォーム送信先の変更
`js/script.js`の`contactForm.addEventListener`セクションで、実際のAPIエンドポイントを設定
